What your AI agent generates.
Without skills vs with skills installed.

✗ Without Skills

# Agent uses Scan — reads entire table response = table.scan( FilterExpression=Attr('customer_id').eq('cust-789') ) # Reads ALL items, pays for every one
→

✓ With Skills

# Agent uses Query — reads only matching partition response = table.query( IndexName='customer-orders-index', KeyConditionExpression=Key('customer_id').eq('cust-789'), ) # 100-1000x cheaper and faster
