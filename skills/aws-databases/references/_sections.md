# Section Definitions

---

## 1. Service Selection (choose)

**Impact:** CRITICAL

**Description:** Choosing the right AWS database service based on access patterns, consistency requirements, latency needs, and operational model.

## 2. Security & Authentication (auth)

**Impact:** CRITICAL

**Description:** IAM database authentication, VPC connectivity, encryption, and security groups across all AWS database services.

## 3. SDK Configuration (sdk)

**Impact:** HIGH

**Description:** AWS SDK client setup with proper retries, timeouts, and connection management for boto3 and AWS SDK v3.
